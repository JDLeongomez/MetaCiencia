## ¿Qué es MetaCiencia?

**MetaCiencia** es un semillero de investigación dedicado a promover prácticas científicas abiertas, reproducibles y transparentes. Nuestros ejes de trabajo son:

- **Reproducibilidad** — condiciones y recursos para que los hallazgos científicos puedan replicarse y auditarse.
- **Ciencia Abierta** — preregistro de estudios, acceso abierto a datos y materiales, y uso de software libre.
- **Metainvestigación** — análisis de cómo se produce, reporta y difunde la ciencia en Colombia y Latinoamérica.
